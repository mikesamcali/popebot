Use Pi’s web search capability to get the current weather in Iran. 1) Query a weather API (e.g., Open-Meteo) for the capital Tehran or a generic location in Iran. 2) Parse the JSON response to extract temperature, weather condition, wind speed, humidity, etc. 3) Format the results as a short human‑readable paragraph and print it to stdout. 4) Optionally write the output to a file `reports/weather-iran.txt` for future reference.  The job should perform the network request, parse the data, and print a concise summary.  No other side‑effects are required.
