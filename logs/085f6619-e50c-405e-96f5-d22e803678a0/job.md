Set up a Telegram connection by creating a new skill at `pi-skills/telegram/`. This skill should have a `SKILL.md` with frontmatter (name: telegram, description: "Send and receive messages via Telegram.") and usage docs. Create a `send.sh` script that takes a message argument, sends it to a Telegram chat via the Telegram API, and make it executable. Activate the skill with `ln -s ../../pi-skills/telegram .pi/skills/telegram`. To test, run `send.sh "test message from thepopebot"` and verify successful delivery. The user needs to obtain a Telegram API token and set it up as an environment variable `TELEGRAM_BOT_TOKEN` using `npx thepopebot set-agent-llm-secret TELEGRAM_BOT_TOKEN <token_value>`, then run this job.
