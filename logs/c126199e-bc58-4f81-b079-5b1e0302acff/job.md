Set up a Telegram connection by registering a Telegram bot and obtaining an API token. Use this token to create a new skill at `pi-skills/telegram/` with a `send.sh` script that takes a message argument and sends it to a Telegram chat via the Telegram API. Activate the skill with `ln -s ../../pi-skills/telegram .pi/skills/telegram`. Test the skill by running `send.sh "test message from thepopebot"` and verify successful delivery. The user needs to set up the Telegram bot token as an environment variable `TELEGRAM_BOT_TOKEN` using `npx thepopebot set-agent-llm-secret TELEGRAM_BOT_TOKEN <token_value>`, then run this job.
